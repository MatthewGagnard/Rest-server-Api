D&D Monster API

store and display stats!

Get all monsters

Example URL: 'https://monster-stat-api.herokuapp.com/Monster'
Results go here!

'/Monster' is used to get every monster we have stored!

Putting "?" after Monster will also allow you to query by certain parameters(CR/HP/AC etc..)

Get Monsters by name

Example URL: 'https://monster-stat-api.herokuapp.com/Monster/containing/<`Name of monster here`>
Results go here!

'/Monster/containg/xxxx' does a case-insensitive search for monster stat blocks.

Create a player!

Example URL: 'https://monster-stat-api.herokuapp.com/Player?NAME=Vensin&HP=45&AC=14&SPEED=30&STR=10&DEX=12&CON=14&WIS=20&INT=10&CHA=8
Results go here!

Create a player character sheet! Currently only availble from here. Fields are all taken after ?. Only accepted "Name,Hp,Ac,Speed,Str,Dex,Con,Int,Wis,Cha, and INVEN (an array of strings containing the inventory)"

Make sure you keep track of the "_id" that comes back. This is your way you access your character!
